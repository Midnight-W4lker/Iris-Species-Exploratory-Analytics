Problem

What the analysis needed to answer

Determine whether four simple flower measurements can reliably separate the three Iris species and identify which variables carry the clearest classification signal.

A transparent research workflow uses descriptive statistics, visual separation analysis, VIF, PCA, ANOVA effect sizes, and regularized multiclass logistic regression.

Findings

The signals that mattered most

  • Petal length and petal width are the strongest species-separation variables.
  • Setosa is clearly distinct, while versicolor and virginica overlap more.
  • Sepal width is the weakest separator and should not be overinterpreted.
  • Petal features are highly correlated, so they are useful but not fully independent.
